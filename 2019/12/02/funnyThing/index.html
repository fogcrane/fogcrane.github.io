前言


由于目前科学上网的软件已是凤毛麟角的存在了，而且还在不断的减少。正所谓靠山山会倒；自己动手，丰衣足食。今天就来说说如何在vultr购买VPS云服务器来搭建SS实现科学上网。

知识储备


什么是Vultr?

vultr是一家创建于2014年的VPS云的商家，坐落于美国新泽西州，其系统可以在人工选择数据中心后60秒内自动部署好VPS，支持API。常规服务器配置为：Intel(R) Xeon(R) CPU E3-1230 v3 @ 3.30GHz，Intel SSD，G口接入，服务器分布于全球6个国家12个数据中心。

什么是VPS?

虚拟专用服务器（英语：Virtual private server，缩写为 VPS），是将一台服务器分区成多个虚拟专享服务器的服务。实现VPS的技术分为容器技术和虚拟化技术 。在容器或虚拟机中，每个VPS都可分配独立公网IP地址、独立操作系统、实现不同VPS间磁盘空间、内存、CPU资源、进程和系统配置的隔离，为用户和应用程序模拟出“独占”使用计算资源的体验。VPS可以像独立服务器一样，重装操作系统，安装程序，单独重启服务器。VPS为用户提供了管理配置的自由，可用于企业虚拟化，也可以用于IDC资源租用。 IDC资源租用，由VPS提供商提供。不同VPS提供商所使用的硬件VPS软件的差异，及销售策略的不同，VPS的使用体验也有较大差异。尤其是VPS提供商超卖，导致实体服务器超负荷时，VPS性能将受到极大影响。相对来说，容器技术比虚拟机技术硬件使用效率更高，更易于超卖，所以一般来说容器VPS的价格都高于虚拟机VPS的价格。 这些VPS主机以最大化的效率共享硬件、软件许可证以及管理资源。每个VPS主机都可分配独立公网IP地址、独立操作系统、独立超大空间、独立内存、独立CPU资源、独立执行程序和独立系统配置等. VPS主机用户除了可以分配多个虚拟主机及无限企业邮箱外， 更具有独立主机功能, 可自行安装程序，单独重启主机。

什么是SS?

shadowsocks是一种基于Socks5代理方式的网络数据加密传输包，并采用Apache许可证、GPL、MIT许可证等多种自由软件许可协议开放源代码。shadowsocks分为服务器端和客户端，在使用之前，需要先将服务器端部署到服务器上面，然后通过客户端连接并创建本地代理。目前包使用Python、C、C++、C#、Go语言等编程语言开发。在中国大陆，本工具也被广泛用于突破防火长城（GFW），以浏览被封锁、屏蔽或干扰的内容。

什么是Xshell 5?

Xshell is a powerful terminal emulator that supports SSH, SFTP, TELNET, RLOGIN and SERIAL. It delivers industry leading performance and feature sets that are not available in its free alternatives. Features that enterprise users will find useful include a tabbed environment, dynamic port forwarding, custom key mapping, user defined buttons, VB scripting, and UNICODE terminal for displaying 2 byte characters and international language support.

科学上网三步曲：

  • 购买VPS服务器
  • 下载安装Xshell 5并连接服务器
  • 搭建 Shadowsocks 服务

购买VPS服务器

首先打开这个链接:我的推荐链接，这里先解释一下所谓的推荐链接，就跟平时见到的邀请开通有相应奖励一样，这个推荐链接也是这么来的。上个图证明一下：

打开页面之后，就是常规的注册账号之类的了，这里说一下充值的问题，第一次充值是需要一次性充值$10，折合RMB就是70元。充值完成之后可以点击侧边栏“Billing”，在右上角就有显示剩余金额。
注册充值完毕，接下来就是如何购买一台VPS服务器了。我选择的是美国Los Angeles的服务器，亲测用了半个月速度还可观，youtube可以看1080P没什么压力。当然这也取决于你自己的网速了。然后是选择CentOS7作为操作系统，套餐选的是$5/月。直接上几张图。

其他几个选项可以按需选择或者直接跳过。选择完毕之后就点击下方的“Deploy Now”。点击后会跳转到此页面，等待即可。

点击进入服务器详情页

记录以下信息，稍后会用到

到这里，恭喜你已经迈出了科学上网的第一步了。接下来的第二步就比较简单了。

下载安装Xshell 5并连接服务器

点击这个链接下载Xshell 5并自行安装，安装过程不懂的话可以自行百度，这里就不赘述了。

安装往之后打开Xshell，进入以下页面，点击新建，然后填入服务器相关信息。

到这里第二步就算完成了，接下来进行第三步。

搭建 Shadowsocks 服务


####1、配置Shadowsocks
首先，安装几个基本的组件。依次输入以下命令，分三次输入，如下图所示。

yum install m2crypto python-setuptools

easy_install pip

pip install shadowsocks

接下来是打开shadowsocks.json文件

vi  /etc/shadowsocks.json

接着按 “i”键进入编辑模式，然后进行如下配置：其中，“server”字段为服务器的IP，“password”字段需要自己进行设置。

{
"server":"0.0.0.0",
"server_port":443,
"local_address": "127.0.0.1",
"local_port":1080,
"password":"123456",
"timeout":300,
"method":"aes-256-cfb",
"fast_open": false
}

编辑后按 “esc” 键退出编辑模式， 输入 “:wq” 保存退出vi。如下图：

####2、配置防火墙
依次输入以下命令，第一条为安装防火墙服务，第二条为开启防火墙服务。

yum install firewalld
systemctl start firewalld

效果如下图：

接下来输入以下命令来开启防火墙端口，其中443端口号可以自定义。

firewall-cmd --permanent --zone=public --add-port=443/tcp
firewall-cmd --reload

效果如下图：

####3、启动Shadowsocks 服务

输入以下命令，让Shadowsocks 服务在后台运行

nohup ssserver -c /etc/shadowsocks.json &

效果如下图：

####4、本地连接ss，实现科学上网

打开ss客户端，输入相应的ip、密码等参数，即可进行科学上网。如下图：
